# History of Deep Learning

## 1943 - First Artificial Neuron

The first mathematical model of an artificial neuron was proposed by:
- Warren McCulloch
- Walter Pitts

This model was inspired by biological neurons in the human brain.

---

# 1958 - Perceptron

Frank Rosenblatt introduced the Perceptron.

A perceptron is the simplest neural network used for binary classification.

Basic idea:

```math
y = wx + b
```

The perceptron could:
- Take inputs
- Apply weights
- Produce output

But it had limitations.

---

# 1969 - AI Winter

Researchers Marvin Minsky and Seymour Papert showed that single-layer perceptrons could not solve complex problems like XOR.

Because of this:
- Interest in neural networks decreased
- Funding reduced
- Research slowed down

This period became known as the AI Winter.

---

# 1986 - Backpropagation Algorithm

Geoffrey Hinton and other researchers popularized Backpropagation.

Backpropagation allowed neural networks to:
- Learn from errors
- Update weights efficiently
- Train deep architectures

This became a major breakthrough in Deep Learning.

---

# 1990s - Rise of Machine Learning

Traditional Machine Learning algorithms became more popular:
- Decision Trees
- SVM
- Naive Bayes

Deep Learning was still limited because:
- Computers were slow
- Data was limited
- GPUs were unavailable

---

# 2006 - Deep Learning Revival

Geoffrey Hinton introduced Deep Belief Networks.

This revived interest in Deep Learning.

Researchers realized that:
- Deep networks could solve complex tasks
- Better hardware could support training

---

# 2012 - AlexNet Revolution

AlexNet won the ImageNet competition with a huge performance improvement.

Key reasons for success:
- GPUs
- Large datasets
- Convolutional Neural Networks (CNNs)
- ReLU activation function

This event changed the AI industry forever.

---

# Biological Neuron vs Artificial Neuron

| Biological Neuron | Artificial Neuron |
|-------------------|------------------|
| Dendrites receive signals | Inputs receive data |
| Cell body processes information | Activation function processes input |
| Axon sends output | Output layer produces prediction |

# Types of Neural Networks

## 1. Artificial Neural Network (ANN)

Basic neural network architecture.

Used for:
- Classification
- Regression

Structure:
- Input Layer
- Hidden Layer
- Output Layer

---

# 2. Convolutional Neural Network (CNN)

Specialized for image processing.

Used in:
- Face Detection
- Object Detection
- Medical Imaging

Key feature:
- Automatically extracts image features.

---

# 3. Recurrent Neural Network (RNN)

Designed for sequential data.

Used in:
- Text Prediction
- Language Translation
- Speech Recognition

RNN remembers previous information using loops.

---

# 4. Long Short-Term Memory (LSTM)

Advanced version of RNN.

Solves:
- Long-term dependency problem

Used in:
- NLP
- Time Series Forecasting

---

# 5. Generative Adversarial Network (GAN)

Introduced by Ian Goodfellow.

Used to generate:
- Fake images
- AI art
- Deepfakes

GAN contains:
- Generator
- Discriminator

---

# 6. Transformer Networks

Most important modern architecture.

Used in:
- ChatGPT
- Gemini
- Claude

Transformers use:
- Attention Mechanism
- Parallel Processing

Advantages:
- Faster training
- Better understanding of language

---

# Feed Forward Neural Network (FFNN)

Simplest neural network where information moves in one direction only.

Flow:

Input → Hidden Layer → Output

No loops or memory present.

---
