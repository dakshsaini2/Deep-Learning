# Day 1 - Introduction to Deep Learning

## What is Deep Learning?

Deep Learning (DL) is a subset of Machine Learning (ML) that uses Artificial Neural Networks with multiple layers to learn patterns from data.

It is inspired by the structure and working of the human brain.

Deep Learning can automatically learn important features from raw data without manual programming.

---

# Artificial Intelligence vs Machine Learning vs Deep Learning

| Artificial Intelligence (AI) | Machine Learning (ML) | Deep Learning (DL) |
|------------------------------|-----------------------|--------------------|
| Broad field of making machines intelligent | Machines learn from data | Uses deep neural networks |
| Includes rule-based systems | Learns patterns using algorithms | Learns complex patterns automatically |
| Parent technology | Subset of AI | Subset of ML |
| Example: Robotics | Spam Detection | Face Recognition |

---
# What is Machine Learning?

Machine Learning is a branch of AI where machines learn from data and improve their performance without being explicitly programmed.

## Types of Machine Learning

### 1. Supervised Learning
Model learns using labeled data.

Examples:
- House Price Prediction
- Email Spam Detection

### 2. Unsupervised Learning
Model learns patterns from unlabeled data.

Examples:
- Customer Segmentation
- Clustering

### 3. Reinforcement Learning
Agent learns using rewards and punishments.

Examples:
- Self Driving Cars
- Game Playing AI

# Neural Networks

Artificial Neural Networks are inspired by biological neurons in the human brain.

## Components of Neural Network

### 1. Input Layer
Receives input data.

### 2. Hidden Layer
Processes information using weights and activation functions.

### 3. Output Layer
Produces final prediction.

---

# Working of a Neural Network

Step-by-step process:

1. Input data is given
2. Data passes through hidden layers
3. Weights are applied
4. Activation functions introduce non-linearity
5. Output is generated
6. Error is calculated
7. Weights are updated using backpropagation

---

# Basic Formula of a Neuron

```math
y = wx + b
```
Where:
- `x` = input
- `w` = weight
- `b` = bias
- `y` = output

---

# Important Deep Learning Terms

| Term | Meaning |
|------|----------|
| Epoch | One complete pass through dataset |
| Batch | Small part of dataset |
| Iteration | One update of model parameters |
| Learning Rate | Controls speed of learning |
| Loss Function | Measures prediction error |
| Activation Function | Adds non-linearity |

# Popular Activation Functions

# Difference Between ML and DL

| Machine Learning | Deep Learning |
|------------------|----------------|
| Requires manual feature engineering | Learns features automatically |
| Performs well on small datasets | Needs large datasets |
| Less computational power | Requires GPUs |
| Faster training | Slower training |
| Easier to interpret | Harder to interpret |
